# CGPA-calculator
A simple, lightweight C++ console application to calculate semester GPA and overall CGPA. Users can input their grades and credit hours to get precise, instant academic results. A clean, efficient command-line tool perfect for students and beginners looking for a quick GPA tracker.

### 💡 Why use this?
Instead of manual calculations, this tool automates the process using standard weighted average formulas. It's clean, fast, and easy to modify for different grading scales (like 4.0 or 5.0 systems).

### 🚀 Key Features
* *Semester GPA:* Calculate performance for a single term by inputting credit hours and grade points.
* *Cumulative CGPA:* Easily track your progress across multiple semesters.
* *Fast & Lightweight:* Zero dependencies; runs directly in your terminal.
